# 🚀 SEO, GEO, AEO 최적화 블로그 포스트 작성 가이드라인

이 가이드는 작성자(Dongju Lee) 님이 향후 블로그 글을 작성할 때, 검색엔진(SEO) 노출을 극대화하고 생성형 AI 검색(GEO) 및 대화형 답변 엔진(AEO)이 콘텐츠를 더 잘 수집·요약하여 출처로 인용할 수 있도록 돕는 스타일 가이드라인입니다.

---

## 1. Markdown 헤더 계층 구조 준수
* **H1(`#`) 헤더 본문 사용 지양**:
  * Jekyll 테마는 기본적으로 포스트 제목을 HTML의 `<h1>` 태그로 자동 지정합니다.
  * 검색엔진 및 AI 크롤러의 시맨틱(Semantic) 웹 페이지 구조 파악을 위해, 본문 내부에서는 H1(`#`)을 사용하지 않고 **H2(`##`)**와 **H3(`###`)**을 사용하여 소제목을 구성해야 합니다.
  * *비추천*: `# 카메라 정보`
  * *추천*: `## 카메라 정보` 또는 `### 세부 사양`

## 2. 도입부 TL;DR (Too Long; Didn't Read) 작성
* **핵심 요약(TL;DR) 배치**:
  * 글의 시작 부분에 전체 내용을 2~3줄로 요약한 핵심 요약 영역을 명시적으로 배치합니다.
  * AI 봇(GPTBot, ClaudeBot 등)이 포스트 전체를 훑지 않고도 핵심 요지를 빠르게 낚아채어 AI 답변 생성의 소스로 사용할 가능성이 크게 상승합니다.
  * *예시 형태*:
    ```markdown
    > **TL;DR**: 이 포스트에서는 Jetson Orin Nano의 GPU 가속 기능(NVDEC)을 활용하여 OpenCV에서 RTSP 실시간 IP 카메라 스트림을 CPU 과부하 없이 웹 브라우저로 송출하는 방법을 정리합니다.
    ```

## 3. 구체적인 이미지 대체 텍스트(alt) 속성 지정
* **의미 있는 alt 설명 제공**:
  * 단순히 `alt="image"` 또는 `alt="스크린샷"` 처럼 작성하면 검색엔진 이미지 검색과 시각장애인용 스크린 리더, 그리고 멀티모달 AI 크롤러가 이미지의 맥락을 이해할 수 없습니다.
  * 이미지가 무엇을 증명하거나 보여주고 있는지 구체적으로 작성하세요.
  * *비추천*: `<img src="..." alt="image" />`
  * *추천*: `<img src="..." alt="jtop 실행 화면에서 NVDEC 비디오 디코딩 영역이 활성화되지 않은 상태" />`

## 4. Q&A 스타일의 제목 및 답변 구조 (AEO 최적화)
* **질문형 헤더 활용**:
  * 사용자가 검색창이나 AI에 질문할 법한 의문형을 헤더(H2, H3)로 채택하고, 그 바로 아래 문단에 간결하고 확실한 직관적 답변을 적어둡니다.
  * 구글의 추천 스니펫(Featured Snippet)이나 Perplexity, ChatGPT Search 등의 직접 답변 엔진에 노출되기 아주 유리합니다.
  * *예시 형태*:
    ```markdown
    ### Q: OpenCV에서 NVDEC 가속이 작동하지 않을 때는 어떻게 해결하나요?
    OpenCV 빌드 시 GStreamer 및 CUDA 연동 옵션(`WITH_GSTREAMER=ON`, `WITH_CUDA=ON`)이 활성화되어 있어야 합니다. `install_opencv.sh` 스크립트를 사용하여 커스텀 빌드 및 재설치를 진행하면 해결할 수 있습니다.
    ```

## 5. 목차(TOC)와 리스트 구조의 활용
* **목록(Bullet/Number) 적극 사용**:
  * 긴 서술형 텍스트보다는 요약 정리할 때 번호 리스트(`1.`, `2.`) 또는 글머리 기호(`-`, `*`)를 적극 활용합니다.
  * AI 답변 엔진은 답변을 요약할 때 목록 형태의 텍스트를 인용하기를 선호합니다.

## 6. 포스트용 FAQ 구조화 데이터(JSON-LD) 추가 (선택사항)
* 특정 포스트가 강한 Q&A 성격을 띠고 있다면, 포스트 마크다운 파일 하단에 아래와 같은 `FAQPage` 스키마 마크업을 삽입해주면 구글 검색 시 리치 스니펫 혜택을 받을 수 있습니다.
  ```html
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    "mainEntity": [{
      "@type": "Question",
      "name": "질문 입력 (예: Jetson Orin Nano에서 NVDEC 사용 방법은?)",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "답변 내용 입력 (HTML 태그 사용 가능)"
      }
    }]
  }
  </script>
  ```
